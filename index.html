About the Program

The Data Science program at the University of Regina offers students a multidisciplinary approach to data science, machine learning, and AI. The program prepares students for industry and academic roles through hands-on projects and advanced research opportunities.

Curriculum (Masters in Data Science)

  • Semester 1: Foundations of Data Science
    • 1. Introduction to Data Science & Programming
      • Overview of data science, ethics, and lifecycle.
      • Programming in Python/R, working with Pandas, NumPy for data manipulation.
    • 2. Statistics & Data Preprocessing
        • o Key statistics (probability, distributions, hypothesis testing). o Data acquisition, cleaning, preprocessing, and feature engineering.
  • Semester 2: Machine Learning and Data Analytics
    • 1. Machine Learning & Data Mining
        • o Supervised/unsupervised learning, algorithms (regression, clustering, decision trees). o Data mining techniques and big data tools like Hadoop and Spark.
    • 2. Data Visualization & Databases
        • o Data visualization (Tableau, Seaborn), storytelling with data. o SQL/NoSQL databases, querying structured/unstructured data.
  • Semester 3: Advanced Machine Learning Techniques
    • 1. Deep Learning & Natural Language Processing (NLP)
      • o Neural networks (CNNs, RNNs), deep learning in image and text analysis. o NLP techniques (sentiment analysis, topic modeling, Word2Vec).
    • 2. Time Series Analysis & Cloud Computing
        • o Forecasting methods (ARIMA, SARIMA) for time-dependent data. o Cloud platforms (AWS, Google Cloud), distributed data processing, big data tools.
  • Semester 4: Specialization and Capstone Preparation
    • 1. Advanced Machine Learning & Specialization
      • o Ensemble methods, dimensionality reduction, model tuning. o Specialization: Healthcare Analytics, Financial Data Science, Marketing Analytics, or Geospatial Data Science.
    • 2. Capstone Project Preparation & Data Ethics
        • o Capstone project planning with real-world problem-solving. o Ethics, privacy, and data security in data science.
  • Semester 5: Internship & Capstone I
    • 1. Industry Internship I
      • Overview of data science, ethics, and lifecycle.
      • Programming in Python/R, working with Pandas, NumPy for data manipulation.
    • 2. Capstone Project I
        • o Key statistics (probability, distributions, hypothesis testing). o Data acquisition, cleaning, preprocessing, and feature engineering.
  • Semester 6: Advanced Internship & Capstone II
    • 1. Industry Internship II
      • Another 4-month internship in a specialized area of data science to deepen industry experience.
    • 2. Capstone Project II & Thesis
        • Finalize and present the capstone project, demonstrating mastery of skills and real-world problem-solving
